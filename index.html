WordPress local development for Mac, Win & Linux

Getting started with lokl


  1. Install Docker
  2. Launch site with 1 command

name=clientsite1;port=4000; docker run -e N=$name -e P=$port --name=$name -p $port:$port -d lokl/local

That's it! In seconds, your WordPress site is all setup and accessible at http://clientsite1.localhost:4000

Features

  • Run 1 or 100's of sites quickly (just use unique name and port in the command!)
  • Publish super fast live sites with bundled plugins
  • Forget about WordPress security headaches
  • Save hosting costs

The WordPress workflow you're used to, but better!

  • Comes pre-loaded with site performance plugins
  • Start fresh or import existing sites easily
  • PHPMyAdmin, web file editor and web SSH access for advanced users
  • 1-click backups to avoid data loss
